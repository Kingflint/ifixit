# Feature catalogue

## Public site

- Hero, Services, How It Works, Features, Contact, Footer sections.
- Repair booking modal with optional category pre-selection.
- Public order tracker keyed off the auto-generated tracking code.

## Auth

- Email + password sign-up, with mandatory email verification.
- Google sign-in, treated as pre-verified.
- An admin-allow-list bypass for the seeded administrator account.

## Admin

- Site settings (business info, fees, feature flags).
- Repair-order management with status transitions.
- Store catalogue and store-order fulfilment.
- Overstay-fee preview when an order has been parked too long.

## Store

- Product browsing with condition, price, and stock filters.
- Cart-style checkout with Paystack payment.
- Purchase history per signed-in user.
